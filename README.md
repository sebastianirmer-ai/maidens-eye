# Maiden's Eye

A lightweight desktop tool to monitor real-time player activity in Demon's Souls (RPCS3).

## Features

- Tracks online players on The Archstones RPCS3 server (https://thearchstones.com)
- Displays total player count and active locations
- Adjustable refresh interval (10 / 30 / 60 seconds)
- Sound notifications on player count changes  
  (high beep = increase, low beep = decrease)

## Quick Start

1. Install Python 3
2. Run `maidens_eye_start.vbs`

## Requirements

- Windows 10/11
- Python 3.7+
- No external dependencies (uses only the Python standard library)
